# 👁️ Machine Vision (Course Repository)

Welcome to the official GitHub repository for the **Machine Vision** course. This repository contains all the course materials, including lecture slides, code examples, assignments, and supplemental resources.

This course bridges the gap between classic geometric computer vision and modern, data-driven deep learning approaches. It is designed to equip students with both the mathematical foundations of image formation and the practical skills to deploy state-of-the-art neural networks.

---

## 📖 Course Syllabus

### Module 1: Traditional Vision

**Theme: Image Formation and Analysis**

* **Camera Model:** Imaging principles, coordinate system transformations, and camera calibration techniques.
* **Stereo Vision:** Binocular vision principles, depth information acquisition, and 3D reconstruction.
* **Feature Extraction and Matching:** Detection algorithms for local features (e.g., SIFT) and robust feature matching methods.

### Module 2: Deep Learning-based Vision

**Theme: Image Classification and Recognition**

* **Neural Network Fundamentals:** Core concepts of artificial neural networks, perceptrons, backpropagation algorithms, and activation functions.
* **Convolutional Neural Networks (CNNs):** In-depth analysis of convolutional layers, pooling layers, fully connected layers, and their advantages in feature extraction.
* **Classic Network Analysis:** Architecture dissection of representative deep learning models (e.g., LeNet, AlexNet, VGG, ResNet).
* **Image Segmentation and Object Detection:** Advanced visual tasks including semantic/instance segmentation, and object detection frameworks (e.g., YOLO series, Faster R-CNN).
